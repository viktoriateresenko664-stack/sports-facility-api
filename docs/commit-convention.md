# Commit Convention

Use Conventional Commits in English.

## Format
`<type>(optional-scope): <short message>`

## Allowed types
- `feat`
- `fix`
- `docs`
- `refactor`
- `test`
- `chore`
- `security`

## Examples
- `feat(auth): add refresh token rotation`
- `fix(ownership): enforce employee task scope`
- `docs(readme): add ngrok integration guide`
- `security(xss): sanitize user text fields`
