# Приложение к документу

## Как запустить
1. `py -3.11 -m venv .venv`
2. `\.venv\Scripts\Activate.ps1`
3. `pip install -r requirements.txt`
4. `alembic upgrade head`
5. `uvicorn app.main:app --host 0.0.0.0 --port 8000`

## API и клиенты
- Единый backend API для всех клиентов.
- Frontend подключается через ngrok URL.
- Swagger: `/docs`.

## CQRS и BFF
- Commands: create/assign/start/finish/cancel/generate report.
- Queries: dashboard/monitoring/list endpoints.
- Read model: агрегированные DTO в BFF.

## Очередь и фоновые задачи
- `command -> domain_event -> queue -> worker`.
- Delayed job: `/reports/generate-delayed` (countdown).
- Статусы задач: `background_jobs`.

## Lifecycle и ownership
- Lifecycle задач: `CREATED -> ACTIVE -> COMPLETED|CANCELLED`.
- USER видит только свои `user_requests`.
- ENGINEER видит только свои `engineer_tasks`.
- OPERATOR/CHIEF_ENGINEER/ADMIN имеют расширенный доступ.

## Security
- JWT/RBAC/ownership validation только на backend.
- Mass assignment запрещен (строгие схемы + owner из JWT).
- XSS sanitation для пользовательских текстовых полей.

## Dev endpoint для Desktop
- `GET /dev/fake-sensor-data?randomize=true`
- `GET /dev/fake-sensor-data?randomize=true&write_to_db=true`
